# Material UI React with Rails

## Installation

- Install Ruby version 3.0.1
- Install Node version 14.17.3
- Install yarn package
- Install gems
    ```
    bundle install
    ```
- Install javascript packages
    ```
    yarn install
    ```
- Run migrations
    ```
    rails db:migrate
    ```
- Run rails server
    ```
    bin/dev
    ```
- URL for the app `http://localhost:3000/`
